## 🏁 Get started
```bash
# Clone this repository
$ git clone https://github.com/lcsdiniz/pokekana-quiz.git

# Go into the repository
$ cd pokekana-quiz

# Install dependencies
$ yarn

# Run the app
$ yarn dev
```

## 🧪 Testing
```bash
# Run automated tests
$ yarn test
```
